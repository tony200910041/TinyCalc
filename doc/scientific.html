TinyCalc: scientific functions


Functions:


TinyCalc supports different scientific functions and constants.
You may enter "function" to obtain a list of function name, and enter "const" to obtain a table of constants.
Functions can be called by the format function_name(argument, argument...), which can be entered in either uppercase or lowercase.
Some functions may return more than one value. See "Memory usage" for further detains".

All methods defined in java.lang.math can be used.
Other methods are listed here:


log
  • argument: the logarithm base b and the parameter a in an expression logba.
  • return: the result of logba

ln
  • argument: the number a that has to be taken a natural logarithm (ln a)
  • return: the result of ln(a)

fact
  • argument: a non-negative integer x being the argument of factorial operation (x!)
  • return: the result of x!

p
  • argument: the total number of elements n, and the number of selected elements r
  • return: the result of nPr (i.e. the number of possible ordered arrangements if r items are selected).

c
  • argument: the total number of elements n, and the number of selected elements r
  • return: the result of nCr (i.e. the number of possible unordered combinations if r items are selected).

sum
  • argument: the lower limit (start), the upper limit (end), and the step parameter
  • the step parameter indicates the common difference between items
  • return: the sum of sequence start, start+step, ... end
  • example: sum(2,5,1) returns 2+3+4+5=14

product
  • argument: the lower limit (start), the upper limit (end), and the step parameter
  • the step parameter indicates the common difference between items
  • return: the product of sequence start, start+step, ... end
  • example: product(3,7,2) returns 3*5*7=105

delta
  • argument: a, b and c in a quadratic function f(x)=ax2+bx+c
  • return: b2-4ac, the discriminant of the equation f(x)=0
  • example: delta(9,25,16) returns 9^2-4*25*16=49

quad
  • argument: a, b and c in a quadratic function f(x)=ax2+bx+c
  • return: an array of results: delta(a,b,c), x1, x2, vert1 and vert2, where x1 and x2 are the real roots of the equation f(x)=0, or the real and imaginary part of the unreal roots of the equation f(x)=0, and (vert1, vert2) is the vertex coordinate of the graph of y=f(x).
  • example: quad(9,25,16) returns {49, -1, -1.7778, -1.3889, -1.3611} (approximately)

heron
  • argument: the three sides of a triangle
  • return: the area of the triangle, calculated by Heron's Formula
  • example: heron(8,5,5) returns 12 (the area of triangle)

opside
  • argument: two of the three sides of a triangle, and the included angle
  • return: the side opposite to the included angle
  • example: opside(3,5,60-d returns 4.3589 (approximately)

area
  • argument: two of the three sides of a triangle, and the included angle
  • return: the area of the triangle
  • example: area(3,5,60-d returns 12.99

angle
  • argument: the three sides of a triangle
  • return: an array of angles, in radians, which are the opposite angles of the three sides respectively
  • example: angle(3,4,5) returns {0.6435, 0.9273, 1.5708} (approximately)

csc
  • argument: an argument a of the expression csc a
  • return: csc a

sec
  • argument: an argument a of the expression sec a
  • return: sec a

cot
  • argument: an argument a of the expression cot a
  • return: cot a

acsc
  • argument: an argument a of the expression acsc a
  • return: acsc a

asec
  • argument: an argument a of the expression asec a
  • return: asec a

acot
  • argument: an argument a of the expression acot a
  • return: acot a

dist
  • argument: the coordinates of two points (x1, y1) and (x2, y2)
  • return: the distance between the two points
  • example: dist(1,1,4,5) returns 5 (the distance between the point (1,1) and the point (4,5))

slope
  • argument: the coordinates of two points (x1, y1) and (x2, y2)
  • return: the slope of the straight line passing through the two points
  • example: slope(1,1,2,3) returns 2

yint
  • argument: the coordinates of two points (x1, y1) and (x2, y2)
  • return: the y-intercept of the straight line passing through the two points
  • example: yint(1,1,2,3 returns -1

distpl
  • argument: the coefficient A, B, C in the equation of straight line Ax+By+C=0, and the two coordinates x, y of a point (x,y)
  • return: the shortest distance (perpendicular distance) between the line and the point
  • example: distpl(3,4,-5,1,2 returns 1.2

det
  • argument: the nine elements, from left to right, top to bottom, in a 3x3 matrix
  • return: the determinant of the matrix

simu2
  • argument: the value a, b, c, d, e, f in a system of linear equation: ax+by=c and dx+ey=f
  • return: an array {x, y}

isprime
  • argument: a integer n, where n>=2
  • return: 1 if n is a prime number, and 0 if not

ran
  • argument: the lower limit and the upper limit
  • return: a random number between the limits

ranint
  • argument: the lower limit and the upper limt, must be integers
  • return: a random integer between the limits


Special conversion:


TinyCalc uses radian as the default angle unit. Currently, TinyCalc only supports radian and degree.
If you want to convert a value between radian and degree, use -r to specify that the original value is in radian, and use -d for degree.

sin(30-d 0.5
acos(0.5)-r 0
You can also type "f", "surd" or "sixty" at the end of the expression to format the result as a fraction, surd or sixty-degree value respectively.

tan(30-dsurd 1/3(sqrt)3
