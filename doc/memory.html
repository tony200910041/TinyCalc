TinyCalc: memory usage


Important notes: All the variables in this part will be saved in a file called TINYCALC_MEMORY which is located at the same directory as the TinyCalc executable.

Answer


The last answer will be saved automatically. Use [ans] to read it.
1+2+3+4+5 (output=15) then [ans]/3 (output=5)
2^3 (output=8) then [ans]^3 (output=512)

Please be reminded that [ans] updates automatically when a calculation result is generated. As such, it is not a reliable variable and may change unexpectedly.

Memory


There are ten memory variables in TinyCalc.
To save a calculation result, use save[*], and to retrieve, use [M*] or [mem*], where * is an integer between 0 and 9 inclusive.
3*4*5save3 (output=60) then 1+[mem3]/5 (output=13)
2^20save9 (output=1048576) then [M9]^(1/10 (output=4)


Temporary variables


There are ten temporary variables in TinyCalc. Unlike memory variables, they can be modified by multi-return methods.
You can use tv* to save a calculation result, and use [tv*], [tmpvar*] or [var*] to retrieve a result, where * is an integer between 0 and 9 inclusive. Note that it is not recommended to use temporary variables to save calculation results as they may be modified.
quad(9,25,16 (output=49) then [tv1] (output=-1) then [tmpvar2] (output=-1.7778) (approximately) then [var3] (output=-1.3889) (approximately)
